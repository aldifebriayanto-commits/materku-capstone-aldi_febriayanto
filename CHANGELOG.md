# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-08

### Added
- ✨ Initial release of Materku
- 📱 Material management (CRUD operations)
- 🗄️ SQLite local database integration
- 📊 Statistics screen with charts (Bar, Pie, Progress)
- ⭐ Favorites system
- 📥 Download tracking
- 🔍 Search and filter functionality
- 🌐 REST API integration for online mode
- ⚙️ Settings screen with online/offline toggle
- 📈 Real-time server health check
- 📝 File picker for material upload
- 🎨 Modern Material Design 3 UI

### Technical
- State management using Provider pattern
- Clean architecture with separation of concerns
- Error handling and user feedback
- Responsive design for various screen sizes
- Performance optimizations
